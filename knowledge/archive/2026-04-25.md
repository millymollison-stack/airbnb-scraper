### Tailwind preflight DISABLED (Apr 25 — could break things)
**File:** `src/tailwind.config.js`
**Change:** Added `corePlugins: { preflight: false }` to disable Tailwind's base reset

**Why:** Tailwind's preflight injects `h1,h2,h3,h4,h5,h6{font-size:inherit;font-weight:inherit}` which was overriding ALL h1 font-size to inherit from parent (making them tiny instead of using our clamp size)

**What this disables (all Tailwind base styles removed):**
- Box-sizing resets
- font-size inheritance on headings (h1-h6)
- margin resets on headings
- list-style resets
- form element base styles
- And many other browser normalizations

**⚠️ RISK:** This could break styled elements elsewhere on the site that relied on Tailwind's preflight. If layout/form styling breaks, re-enable preflight and keep the !important h1 workaround instead.
