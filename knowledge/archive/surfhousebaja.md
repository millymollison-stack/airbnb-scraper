# Surfhousebaja Project

## Status: Active

## Features Implemented
- Property listing with image gallery
- Booking calendar
- Amenities section with 50 options
- Edit mode for admin users
- Reviews system
- Sidebar with profile management
- Custom domain support
- Stripe payout configuration

## Technical Details
- Frontend: React + Vite + Tailwind
- Backend: Supabase
- Deployment: Netlify/Vercel

## Recent Changes (March 16, 2026)

- Stat lozenges: 3 columns at 640px
- Intro textbox: 2 rows, moved up 10px
- Profile title: hero-title font, black text
- Review admin buttons: circular, matching sizes
- Sidebar: large drop shadow
