# Nightly Consolidation Task

This runs at 2AM each night.

## Steps:
1. Read today's chat sessions from sessions_history
2. Extract key learnings and important information
3. Update /knowledge/daily/YYYY-MM-DD.md with learnings
4. Update relevant project notes in /knowledge/projects/
5. Move any completed items to /knowledge/archive/
6. Reply with summary of what was updated

## Priority Information to Capture:
- Project updates & progress
- Decisions made
- Code changes & features
- Problems solved
- Important insights
- Contacts & relationships
- Financial updates
